# Versión en español 

## ⚛️ Mis primeros proyectos de Computación Cuántica

Este repositorio contiene mis avances, experimentos y ejercicios de aprendizaje en el mundo de la computación cuántica. Aquí exploro diferentes arquitecturas y algoritmos utilizando los frameworks más destacados de la industria y la investigación.

### 🚀 Frameworks en uso
* **PennyLane:** Enfoque en Quantum Machine Learning (QML) y diferenciación de circuitos.
* **Qiskit:** Implementación de algoritmos cuánticos estándar y computación basada en puertas.
* *(Próximamente)* **Cirq / Braket:** Otros frameworks de interés para experimentación.

### 📂 Contenido del repositorio
* `/pennylane_tutorial.ipynb`: Introducción a circuitos variacionales y optimización.
* `/QML_introduccion`: Notebooks y scripts enfocados en la intersección de ML y Quantum.
* `main.py`: Saludo automático implementado por uv.

### 🛠️ Configuración del entorno
Este proyecto utiliza **uv** para la gestión de dependencias y entornos virtuales.

Para replicar el entorno:
```bash
# Instalar dependencias
uv sync

# Activar el entorno
source .venv/bin/activate
```

### 📚 Notas 
* Este repositorio es un espacio de aprendizaje y experimentación, por lo que el código puede ser un work in progress.
* Se recomienda revisar los notebooks para entender el contexto y los objetivos de cada experimento.

Repositorio creado por Jorge Toral. ¡Bienvenidos a este viaje por la computación cuántica!

# English version
## ⚛️ My First Quantum Computing Projects
This repository contains my progress, experiments, and learning exercises in the world of quantum computing. Here I explore different architectures and algorithms using the most prominent frameworks in the industry and research.

### 🚀 Frameworks in use
* **PennyLane:** Focus on Quantum Machine Learning (QML) and circuit differentiation.
* **Qiskit:** Implementation of standard quantum algorithms and gate-based computing.
* *(Coming soon)* **Cirq / Braket:** Other frameworks of interest for experimentation.

### 📂 Repository content
* `/pennylane_tutorial.ipynb`: Introduction to variational circuits and optimization.
* `/QML_introduccion`: Notebooks and scripts focused on the intersection of ML and Quantum.
* `main.py`: Automatic greeting implemented by uv.

### 🛠️ Environment setup
This project uses **uv** for dependency management and virtual environments.
To replicate the environment:
```bash
# Install dependencies
uv sync

# Activate the environment
source .venv/bin/activate
```
